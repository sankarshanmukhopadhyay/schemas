# dtg-credentials (upstream reference)

- Upstream repo: https://github.com/LF-Decentralized-Trust-labs/dtg-credentials

## Why it matters here

`dtg-credentials` defines credential schemas and semantics used by DTG Labs ecosystem implementations.
This repo set focuses on **assurance and conformance artifacts** that can be applied to those credential families.

## Interoperability touchpoints

- Map credential families to applicable conformance profiles and evidence expectations.
- Ensure schema identifiers and versions are pinned and traceable across releases.
