# verifiable-trust-infrastructure (upstream reference)

- Upstream repo: https://github.com/LF-Decentralized-Trust-labs/verifiable-trust-infrastructure

## Why it matters here

This repo describes reference architecture and components for verifiable trust ecosystems.
This repo set provides the **assurance overlay**: controls, evidence bundles, and evaluation workflows.

## Interoperability touchpoints

- Use VTI architecture as the system boundary definition for assurance profiles.
- Align roles (issuer, verifier, registry operator, governance authority) with DCAS profiles and evidence catalogs.
