# v0.4.0

This release introduces **transport alignment** guidance with the ToIP **Trust Spanning Protocol (TSP)** and strengthens repository hygiene for long-term maintainability.

## Highlights

- Added transport alignment documentation for TSP (non-normative).
- Updated architecture snapshot to include the transport layer and fixed Mermaid syntax for GitHub rendering.
- Added Dependabot configuration for GitHub Actions updates.
- Hardened GitHub Actions workflows for deterministic runs and clearer failures.

## Notes

- This release does not introduce new credential semantics; it improves interoperability documentation and operational reliability.
